# User Configs

`config/users/` is reserved for per-user runtime configs generated by the chat API auth system.

Generated layout per user:

- `config/users/<username>/chat_controller.yaml`
- `config/users/<username>/memory_agent.yaml`
- `config/users/<username>/memory_core.yaml`

User account metadata is stored in:

- `config/users/users.json`

Do not commit real credentials or production user files.
