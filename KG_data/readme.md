## 知识图谱的定义
1. 关于认识：认识可以分为多个层级，由浅到深，

* 自我认知图谱：存储个人自我概念、自尊、自我效能等自我相关的认知和评价数据。
* 人际关系图谱：存储个人社会网络、亲密关系、角色关系、互动历史等社会关系数据。
* 社会群体图谱：存储群体成员身份、社会身份、群体规范、群体动态等群体相关数据。
* 社会认知图谱：存储对他人和社会的认知结构，如刻板印象、归因、态度、信念等数据。
* 文化规范图谱：存储社会文化规范、价值观、习俗、道德准则等共享文化知识数据。
* 情景记忆图谱：存储个人经历的具体事件，特别是与社会互动相关的情景记忆数据。
* 情绪情感图谱：存储个人情绪体验、情感反应、情绪调节等情绪相关数据。
